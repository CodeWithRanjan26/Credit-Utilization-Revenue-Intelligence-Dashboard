# 💳 Credit Utilization & Revenue Intelligence Dashboard

## 🚀 Overview

This project is an end-to-end **Decision Intelligence Dashboard** built using Power BI, SQL, and Python.

It focuses on analyzing customer credit behavior to generate **actionable business insights** related to:

* 📈 Revenue Growth
* ⚠️ Risk Management
* 🎯 Customer Segmentation
* 🧠 Decision Making

---

## 🎯 Objective

The goal of this project is to:

* Identify **high-value customers**
* Detect **risky customers**
* Optimize **credit limit allocation**
* Improve **customer targeting strategies**
* Enable **data-driven decision systems**

---

## 🛠️ Tools & Technologies

* 📊 Power BI (Dashboard & Visualization)
* 🗄️ SQL (Data Analysis & Querying)
* 🐍 Python (Data Cleaning & Preprocessing)

---

## 📂 Dataset

* Dataset: Credit Card Customer Dataset
* Records: 30,000+ customers
* Features include:

  * Credit Limit
  * Bill Amounts
  * Payment History
  * Customer Demographics
  * Default Behavior

📌 Due to size limitations:

* Dataset & Power BI File available here 👉 **[Google Drive Link]**

## 🔄 Project Workflow

1. Data Cleaning using Python
2. Feature Engineering
3. Data Analysis using SQL
4. Data Modeling in Power BI
5. Dashboard Creation
6. Insight & Decision Building
